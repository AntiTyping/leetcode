    346. Moving Average from Data Stream EasySliding Window · 11 days since review · rep 2, interval 6

    The Problem

    Design a class that calculates the moving average of the last size values seen in a stream. Each call to next(val) adds a value and returns the average of the most recent (up to) size values.

    Quick Reminder

    Key InsightInstead of recalculating the sum each time, maintain a running sum — subtract the element leaving the window, add the new one. That makes each call O(1).

    class MovingAverage(object):
    def __init__(self, size):
        self.q = []
        self.size = size
        self.sum = 0
    def next(self, val):
        self.sum += val
        self.q.append(val)
        if len(self.q) > self.size:
            x = self.q.pop(0)

    3. Where they meet again is the cycle entrance — return that value (the duplicate).

    Why This WorksMapping index→value forms a functional graph where every node has out-degree 1. With a repeated value, two nodes share a successor, forcing a cycle.     Watch Out ForThe classic bug is forgetting the double-step for fast in phase 1. Phase 2 pointers move ONE step each, not two — using two steps there breaks the proof.
